## Overview
This is a **Full Stack Development project** simulating an **admin panel for online service bookings**, invoice management, and Accounts Receivable (AR) workflows.  
Built using **React**, **Node.js**, and **MySQL**, this project is relevant for **billing and AR processes**.

---

## Features
- Track and manage online service bookings  
- Generate invoices automatically  
- Validate booking data (placeholder scripts)  
- Dashboard for recurring billing simulation  
- Handle customer service requests
